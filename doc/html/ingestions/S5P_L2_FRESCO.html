S5P_L2_FRESCO

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of S5P_L2_FRESCO data.

field name type dimensions unit description
index int32 {time}   zero-based index of the sample within the source product
scanline_pixel_index int16 {time}   pixel index (0-based) within the scanline
datetime double {time} seconds since 2000-01-01 start time of the measurement
longitude float {time} degree_east longitude of the ground pixel center (WGS84)
latitude float {time} degree_north latitude of the ground pixel center (WGS84)
cloud_fraction float {time}   effective cloud fraction
cloud_fraction_stdev float {time}   effective cloud fraction precision (1 sigma error)
cloud_pressure float {time} hPa cloud pressure, at approximately the mid-level of the cloud layer
cloud_pressure_stdev float {time} hPa cloud pressure precision (1 sigma error)
cloud_height float {time} m cloud height, at the optical centroid level, measured from the surface
cloud_height_stdev float {time} m cloud height precision (1 sigma error)
cloud_albedo float {time}   cloud albedo; this is a fixed value for FRESCO
cloud_albedo_stdev float {time}   cloud albedo error; since cloud albedo is fixed for FRESCO, this value is set to NaN
scene_albedo float {time}   scene albedo when FRESCO is running in snow/ice mode (this quantity is required by the CH4 processor)
scene_albedo_stdev float {time}   scene albedo precision (1 sigma error) when FRESCO is running in snow/ice mode
scene_pressure float {time} hPa apparent scene pressure when FRESCO is running in snow/ice mode (this quantity is required by the CH4 processor)
scene_pressure_stdev float {time} hPa apparent scene precision (1 sigma error) when FRESCO is running in snow/ice mode
