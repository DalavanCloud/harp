Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of SMR_L2 data.

field name type dimensions unit description
datetime double {time} seconds since 2000-01-01 The time of the measurement
latitude_per_profile double {time} degree_north The center latitude for a profile
longitude_per_profile double {time} degree_east The center longitude for a profile
solar_zenith_angle_per_profile double {time}   Average solar zenith angle for the scan
altitude double {time, vertical} km The altitude in km for each profile element
ClO_volume_mixing_ratio double {time, vertical}   ClO volume mixing ratio
ClO_volume_mixing_ratio_uncertainty double {time, vertical}   Precision of the ClO volume mixing ratio

HNO3_volume_mixing_ratio double {time, vertical}   HNO3 volume mixing ratio

BrO2_volume_mixing_ratio double {time, vertical}   BrO2 volume mixing ratio
BrO2_volume_mixing_ratio_uncertainty double {time, vertical}   Precision of the BrO2 volume mixing ratio
temperature double {time, vertical} K Temperature
temperature_uncertainty double {time, vertical}   Precision of the temperature
pressure double {time, vertical} K Pressure
pressure_uncertainty double {time, vertical}   Precision of the pressure
index int32 {time}   zero-based index of the sample within the source product

The table below details where and how each variable was retrieved from the input product.
