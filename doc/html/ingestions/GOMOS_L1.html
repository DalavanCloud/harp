GOMOS_L1

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of GOMOS_L1 data.

field name type dimensions unit description
datetime double {time} seconds since 2000-01-01 The time of the measurement
latitude double {time} degree_north The latitude of the apparent tangent point
longitude double {time} degree_east The longitude of the apparent tangent point
altitude double {time} km The altitude of the apparent tangent point in km
spectral_photon_irradiance double {time, spectral} ph/s/cm^-2/nm The spectral photon irradiance of each spectrum measurement
spectral_photon_irradiance_error double {time, spectral} ph/s/cm^-2/nm The error in the spectral photon irradiance of each spectrum measurement
wavelength double {spectral} nm The nominal wavelength assignment for each of the detector pixels
datetime_length double   s The integration time for a readout (in seconds)
sensor_latitude double {time} degree_north The latitude of the satellite position at half-measurement
sensor_longitude double {time} degree_east The longitude of the satellite position at half-measurement
sensor_altitude double {time} km The altitude of the satellite at half-measurement
flag_illumination_condition string     The illumination condition for each profile: ‘dark’, ‘bright’, ‘twilight’, ‘straylight’ or ‘twilight/straylight’
index int32 {time}   zero-based index of the sample within the source product

Mapping description

The table below details where and how each variable was retrieved from the input product.

GOMOS Level 1 products only contain a single profile; all measured transmission data will be provided in order from high altitude to low altitude in the profile

field name mapping description
datetime path /tra_transmission/dsr_time
latitude path /tra_geolocation/tangent_lat[1]
longitude path /tra_geolocation/tangent_long[1]
altitude path /tra_geolocation/tangent_alt[1]
description The value is divided by 1000 to convert it to km
spectral_photon_irradiance path /tra_transmission[]/trans_spectra[]
spectral_photon_irradiance_error path /tra_transmission[]/cov[]
wavelength path /tra_nom_wav_assignment[]/nom_wl[]
description Will be set to nm
datetime_length path /sph/samp_duration
sensor_latitude path /tra_geolocation/lat[1]
sensor_longitude path /tra_geolocation/longit[1]
sensor_altitude path /tra_geolocation/alt[1]
description The value is divided by 1000 to convert it to km
flag_illumination_condition condition CODA product version 0
path /tra_summary_quality[0]/limb_flag
condition CODA product version 1 and higher
path /tra_summary_quality[0]/obs_illum_cond
