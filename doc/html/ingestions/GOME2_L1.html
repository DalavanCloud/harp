GOME2_L1

Variables

The table below lists the variables that are present in the HARP product that results from an ingestion of GOME2_L1 data.

field name type dimensions unit description
datetime_start double {time} seconds since 2000-01-01 start UTC date/time of Sun calibration mode measurement
datetime_end double {time} seconds since 2000-01-01 end UTC date/time of Sun calibration mode measurement
sun_reference_wavelength_photon_irradiance double {time, spectral} count/s/cm2/nm solar mean reference spectrum
datetime double {time} seconds since 2000-01-01 time of the measurement at the end of the integration time
latitude double {time} degree_north center latitude of the measurement
longitude double {time} degree_east center longitude of the measurement
latitude_bounds double {time, 4} degree_north corner latitudes for the ground pixel of the measurement
longitude_bounds double {time, 4} degree_north corner longitudes for the ground pixel of the measurement
wavelength_photon_radiance double {time, spectral} count/s/cm2/sr/nm measured radiances
transmittance double {time, spectral}   transmittance
wavelength_photon_irradiance_sun double {time, spectral} count/s/cm2/nm measured sun irradiances
wavelength_photon_irradiance_moon double {time, spectral} count/s/cm2/nm measured moon irradiances
wavelength double {time, spectral} nm nominal wavelength assignment for each of the detector pixels
datetime_length double {time, spectral} s integration time for each pixel
scan_subset_counter int8 {time}   relative index (0-3) of this measurement within a scan (forward+backward)
scan_direction string {time}   scan direction for each measurement: ‘forward’, ‘backward’ or ‘mixed’ (for a measurement that consisted of both a forward and backward scan)
cloud_top_pressure double {time}   cloud top pressure
cloud_fraction double {time}   cloud fraction
solar_zenith_angle_toa double {time}   solar zenith angle at top of atmosphere
solar_azimuth_angle_toa double {time}   solar azimuth angle at top of atmosphere
viewing_zenith_angle_toa double {time}   viewing zenith angle at top of atmosphere
viewing_azimuth_angle_toa double {time}   viewing azimuth angle at top of atmosphere
index int32 {time}   zero-based index of the sample within the source product

Ingestion options

The table below lists the available ingestion options for GOME2_L1 products.

option name legal values description
band band-1a, band-1b, band-2a, band-2b, band-3, band-4 only include data from the specified band (‘band-1a’, ‘band-1b’, ‘band-2a’, ‘band-2b’, ‘band-3’, ‘band-4’); by default data from all bands is retrieved
data radiance, transmission, sun, moon, sun_reference retrieve the measured radiances, the transmission spectra, the sun measurement spectra, the moon measurement spectra or the sun reference spectrum; by default the measured radiances are retrieved
