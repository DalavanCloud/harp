Dataset

group harp_dataset

The HARP harp_dataset module contains everything regarding HARP datasets.

Typedefs

typedef struct harp_dataset_struct harp_dataset

HARP Dataset typedef

Functions

int harp_dataset_new(harp_dataset **new_dataset)

Create new HARP dataset. The metadata will be intialized with zero product metadata elements.

Return
  • 0, Success.
  • -1, Error occurred (check harp_errno).
Parameters
  • new_dataset -

    Pointer to the C variable where the new HARP product metadata will be stored.

void harp_dataset_delete(harp_dataset *dataset)

Delete HARP dataset.

Parameters
  • dataset -

    Pointer to the dataset to free.

void harp_dataset_print(harp_dataset *dataset, int (*print)(const char *, ...))

Print HARP dataset.

Parameters
  • dataset -

    Pointer to the dataset to print.

  • print -

    Pointer to the function that should be used for printing.

int harp_dataset_add_directory(harp_dataset *dataset, const char *pathname)

Read metadata for all products found in the directory indicated by path.

Return
  • 0, Success.
  • -1, Error occurred (check harp_errno).
Parameters
  • pathname -

    Path of the directory to search for products to import as a HARP harp_dataset.

  • dataset -

    Pointer to dataset to add the metadata to.

int harp_dataset_add_file(harp_dataset *dataset, const char *filename)

Read metadata for product indicated by filename and add it to the dataset. This will not check if filename already appears in the dataset.

Return
  • 0, Success.
  • -1, Error occurred (check harp_errno).
Parameters
  • filename -

    Path of the directory to search for products to import as a HARP harp_dataset.

  • dataset -

    Pointer to dataset to add the metadata to.

long harp_dataset_get_index_from_source_product(harp_dataset *dataset, const char *source_product)

Lookup the index of source_product in the given dataset.

Return
index of source_product in dataset or -1 if not found
Parameters
  • dataset -

    Pointer to dataset to get index in.

  • dataset -

    Pointer of product to get index of.

int harp_dataset_add_product(harp_dataset *dataset, const char *source_product, harp_product_metadata *metadata)

Add a product to a dataset. Metadata is only allowed to be NULL if the dataset does not track metadata.

Return
  • 0, Success.
  • -1, Error occurred (check harp_errno).
